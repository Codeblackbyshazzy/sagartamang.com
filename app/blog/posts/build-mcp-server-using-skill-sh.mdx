## 🤖 What Is an MCP Server (and Why Should You Build One)?

**MCP (Model Context Protocol)** is an open standard that lets AI agents like Claude connect to external tools and data sources. When you wrap an API inside an MCP server, your agent can call it as a native tool — no copy-pasting, no tab switching, no manual prompting with raw data.

Examples of what you can build:

- A **Groww MCP server** so Claude can query your portfolio and place trades
- A **Notion MCP server** so your agent can read and write your notes
- A **custom REST API MCP server** so your internal tools become AI-native

The only thing stopping most developers is the setup time. The MCP Builder Skill eliminates that.

## ⚡ The MCP Builder Skill — What It Does

The **MCP Builder Skill** is a pre-packaged skill you install in your IDE that teaches your coding agent exactly how to scaffold MCP servers — in Python or TypeScript. Once installed, you just describe the API you want to wrap, and the agent handles the rest.

## 🛠️ Step-by-Step: Build an MCP Server in Under a Minute

### Step 1: Get the MCP Builder Skill

Head to **[skills.sh](https://skills.sh)** and search for **`mcp-builder`**.

Copy the install command from the skill's page. It will look something like:

```bash
npx skills-cli install mcp-builder
```

### Step 2: Open Your IDE

Open your favourite IDE — VS Code, Cursor, Windsurf, or anything with an integrated terminal. Navigate to the project folder where you want to scaffold your MCP server.

### Step 3: Run the Install Command

Open the terminal and paste the command you copied from skills.sh:

```bash
npx skills-cli install mcp-builder
```

That's it. Your agent now knows how to build an MCP server in **Python or TypeScript** — framework conventions, tool registration, SSE transport, the works.

### Step 4: Instruct Your Agent to Scaffold the API

Now just tell your agent what you want. Be specific about the API you're wrapping. For example:

> "Scaffold all the Groww Python API endpoints as MCP tools. Use the Groww Python SDK. Expose portfolio, positions, watchlist, and order placement as separate tools."

The agent will generate the full MCP server structure — tool definitions, argument schemas, error handling, and a working `main.py` or `index.ts` entry point.

### Step 5: Configure Your API Keys and Run

Drop your API credentials into the generated `.env` file:

```env
GROWW_API_KEY=your_api_key_here
GROWW_API_SECRET=your_api_secret_here
```

Then start the server:

```bash
# Python
uv run main.py

# TypeScript
npx ts-node index.ts
```

Your MCP server is now live at `http://127.0.0.1:8000`.

### Step 6: Connect to Claude Desktop

Open your Claude Desktop config (`claude_desktop_config.json`) and add:

```json
"mcpServers": {
  "your_mcp_name": {
    "command": "npx",
    "args": [
      "-y",
      "mcp-remote",
      "http://127.0.0.1:8000/sse"
    ]
  }
}
```

Restart Claude Desktop, and your new MCP server is connected as a native tool.
